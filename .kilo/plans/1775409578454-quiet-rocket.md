# Plan : Test des phrases avec apostrophes du Chapitre 2

## Objectif
Extraire **toutes** les phrases contenant des apostrophes du fichier `Chapitre 2 _ La Part du Vide.md`, générer un fichier audio pour chaque phrase complète, et valider les résultats avec l'utilisateur.

## Contexte
- Le projet dispose déjà de `fix_french_elisions_ssml()` dans `utils.py` pour corriger les élisions
- Un script de test existe: `test_elisions_chapitre2.py`
- Les phrases de test sont dans `phrases_chapitre2.py`

## Choix utilisateur
- **Nombre de phrases**: Toutes les phrases avec apostrophes (~100+)
- **Longueur**: Phrases complètes (pas d'extraits)

## Étapes d'implémentation

### 1. Créer un script d'extraction (`extract_phrases_apostrophes.py`)
```python
- Lire le fichier Markdown
- Découper en phrases (regex sur ponctuation)
- Filtrer les phrases contenant des élisions: [a-zA-Z]'[aeiouéèêàâîïôûùh]
- Sauvegarder dans un fichier Python (liste PHRASES)
```

### 2. Créer un script de génération audio (`test_all_elisions.py`)
```python
- Importer la liste PHRASES
- Pour chaque phrase:
  - Appliquer fix_french_elisions_ssml()
  - Générer le SSML
  - Appeler Google TTS
  - Sauvegarder en MP3: test_elision_001.mp3, test_elision_002.mp3, etc.
- Afficher un récapitulatif avec le texte et le SSML généré
```

### 3. Exécuter et valider
- Lancer le script de génération
- Présenter les fichiers à l'utilisateur pour écoute
- Collecter les retours sur la qualité de prononciation

## Types d'élisions à tester
| Élision | Exemple | Fréquence estimée |
|---------|---------|-------------------|
| d' | d'adultes, d'un | ~40% |
| l' | l'objet, l'air | ~25% |
| n' | n'était, n'a | ~15% |

| c' | c'était, c'est | ~5% |

| m' | m'a | ~1% |
